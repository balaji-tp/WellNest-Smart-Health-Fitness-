# WellNest – Smart Health & Fitness System

## Overview
WellNest is a Smart Health and Fitness Monitoring System designed to track important health parameters in real time using IoT technology. It helps users monitor their health and maintain a better lifestyle.

---

## Features
- Real-time health monitoring  
- Heart rate measurement  
- Body temperature tracking  
- Activity and step monitoring  
- Wireless data transmission  
- Cloud data storage  
- Easy-to-use dashboard  

---

## Technologies Used

### Hardware
- Microcontroller (Arduino / ESP32)
- Heart Rate Sensor
- Temperature Sensor
- Motion Sensor
- WiFi Module

### Software
- Arduino IDE
- Embedded C
- IoT Cloud Platform

---

## How It Works
1. Sensors collect health data from the user.
2. Microcontroller processes the data.
3. Data is sent to the cloud using WiFi.
4. Users can view their health details on a dashboard.

---

## Applications
- Personal health monitoring  
- Fitness tracking  
- Remote patient care  
- Elderly health support
